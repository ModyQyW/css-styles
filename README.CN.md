# css-styles

我在不同项目中使用的 css 样式集合，包括 react-native，小程序（taro / uni-app）和网页（react / vue）

[English](README.md) | 简体中文

## 起步

复制到你的项目中，引用，然后写上 class 或者 className。

## 测试

目前没有测试。欢迎 PR。

## 特性

- 预设布局
- 预设排版
- 两种风格的预设颜色：Material Design 和 Ant Design
